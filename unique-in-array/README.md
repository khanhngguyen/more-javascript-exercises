Write a function that, given a array, return a new array where each element only appears once

For example:
- [ 1, 2, 1, 2, 3 ] => [ 1, 2, 3 ]
- [ "Scooby", "Dooby", "Dooby", "Doo" ] => [ "Scooby", "Doo" ]
- [ "DS", "Algo", "OS", "HTML", "DS", "OS", "Java", "HTML", "Algo" ] => [ "DS", "Algo", "OS", "HTML", "Java" ]
